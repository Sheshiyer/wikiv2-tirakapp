# Visual Assets

Welcome to the central repository for the Tirak Dream Journeys campaign's visual assets. This directory includes high-fidelity catalog layouts, campaign grids, and contact sheets created to guide the visual direction of our cross-channel marketing efforts.

## Campaign Grids & Catalog Layouts

These assets demonstrate how our visual identity comes to life in structured layouts. They provide a reference for formatting content across different mediums, ensuring consistency in how we present the Tirak experience.

## Usage Guidelines

- Always use the WebP format for static images to ensure optimal loading performance.
- When placing images in UI components like `<AdAssetGrid>`, use the `image` prop (e.g., `image: '/images/1.webp'`).
- Ensure all images have descriptive `alt` text for accessibility.
