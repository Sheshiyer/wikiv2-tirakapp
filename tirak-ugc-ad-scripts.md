## 📋 PRODUCTION NOTES

### UGC Style Guidelines
- **Camera:** iPhone/Samsung front-facing camera. No stabilizers. Slight shake = authentic.
- **Lighting:** Natural light or available light. Never studio. Golden hour preferred for outdoor shots.
- **Wardrobe:** Whatever the person would actually wear. No branded clothing.
- **Editing:** Quick cuts, no fancy transitions. Text overlays in bold sans-serif (white with black shadow). Subtitles mandatory.
- **Music:** Use trending sounds from TikTok/Reels. Each script notes a vibe — match to whatever's trending at time of production.
- **CTA:** Always "Link in bio" — drives to Tirak landing page / app store.

### Casting Notes
- **Companion roles (Intro 1, Wellness, Lifestyle, Food):** Cast real Thai locals age 24-30. Must speak natural English with Thai accent (NOT polished). Authentic > perfect.
- **Traveler roles (Intro 2, Culture, Adventure, Nightlife, Private, Cinema, Events):** Cast real Western travelers/nomads age 27-35. Diverse backgrounds. Must feel genuinely excited, not scripted.
- **Priority:** If possible, use ACTUAL Tirak companions and travelers, not actors. Real testimonials outperform actors 3-5x on social.

### Viral Optimization
| Element | Strategy |
|---------|----------|
| **Hook** | Pattern interrupt in first 2 seconds. Shocking stat, call-out, or visual reveal. |
| **Retention** | New visual every 2-3 seconds. Never stay on one shot for more than 4 seconds. |
| **Shareability** | Each script has a "quotable moment" designed for screenshots/shares. |
| **Comments bait** | Controversial opinions ("Khao San Road," "airport Subway") drive engagement. |
| **Save trigger** | Practical value (price comparisons, hidden spots) drives saves — the algorithm's favorite signal. |
| **Hashtags** | #tirak #bangkoktravel #thingstodinbangkok #hiddenthailand #travelcompanion #notaTour #digitalnomad |

### Brand Voice Checklist (Per Script)
- [ ] Uses "companion" never "tour guide"
- [ ] Uses "vibe" / "real" / "verified" / "authentic"
- [ ] Avoids "exotic" / "mystical" / "curated experiences"
- [ ] Mentions specific commission (85%) or price comparison where relevant
- [ ] Feels like a friend talking, not a brand advertising
- [ ] Has ONE clear emotion: excitement, relief, empowerment, or FOMO
- [ ] CTA is always "Link in bio" (not "download now" or "sign up")
